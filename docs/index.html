Why Dasyl?

Everything you need to go from zero to a running project — in seconds.

Quick Scaffolding

Spin up a fully-structured project with a single command. No more copying boilerplate by hand.

TypeScript Ready

First-class TypeScript support for Node.js projects — tsconfig, path aliases, and compiled output all pre-configured.

Auto Installation

Optionally install all npm dependencies right after scaffolding — zero extra steps required.

Git Integration

Initialize a Git repository and commit the initial scaffold automatically so you start with a clean history.

IDE Support

Open the new project in VS Code instantly — no manual code . needed.

Cross-Platform

Works seamlessly on Windows, macOS, and Linux — including proper npm.cmd detection on Windows.

Quick Start

Install once, scaffold forever.

1

Install globally

npm install -g dasyl
2

Run a quick shortcut — or use interactive mode

# React app (Vite)
dasyl react my-app

# Node.js Express (JavaScript)
dasyl node my-api

# Node.js Express (TypeScript)
dasyl node-ts my-api

# Laravel project
dasyl laravel my-laravel-app
# Launch the interactive wizard
dasyl

# Follow the prompts:
#  › Project name?
#  › Frontend or Backend?
#  › Framework?
#  › Install dependencies? (y/n)
#  › Init Git repo? (y/n)
#  › Open in VS Code? (y/n)
3

Use flags to skip prompts

# Accept all defaults in one shot
dasyl react my-app --yes

# Skip installing dependencies
dasyl node my-api --skip-install

# Skip Git init
dasyl node my-api --skip-git

# Create in a custom directory
dasyl react my-app --dir /path/to/projects
4

Start building

Your project is ready. Dependencies installed, Git initialized, and VS Code open — straight to coding.
