Quick Start

Install once, scaffold forever.

1

Install globally

npm install -g dasyl
2

Run a quick shortcut - or use interactive mode

# React app (Vite)
dasyl react my-app

# Node.js Express (JavaScript)
dasyl node my-api

# Node.js Express (TypeScript)
dasyl node-ts my-api

# Laravel project
dasyl laravel my-laravel-app
# Launch the interactive wizard
dasyl

# Follow the prompts:
#  > Project name?
#  > Frontend or Backend?
#  > Framework?
#  > Install dependencies? (y/n)
#  > Init Git repo? (y/n)
#  > Open in VS Code? (y/n)
3

Start building

Your project is ready. Dependencies installed, Git initialized, and VS Code open - straight to coding.

Backend

Node.js Express

Production-ready REST API boilerplate with Express, MongoDB, and auth.

  • [!] Helmet + CORS security
  • [!] Auth routes & User model

  • [!] MongoDB / Mongoose ready
  • [!] Controllers / Routes / Models

PHP

Laravel

Scaffold a fresh Laravel project with pre-configured code quality tools and API scaffolding.

  • [!] Latest Laravel version
  • [!] API scaffolding installed
